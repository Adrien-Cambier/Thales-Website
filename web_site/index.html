Sniffer Ethernet

Extract Frames Python Script

First, run the python script to generate a database.
The aim is to display Ethernet frames from functional validation tests on a space module by decoding part of the frames.

Explore Data Web Interface

Then, explore data using a nice web interface.
You can search for binary records by name and date, and modify their names.

What it allows you in few words

EXTRACT FRAMES

It starts by extracting all the frames

WRITE DATABASE

Then it puts the frames in a SQL database

EXPLORE DATA

You explore data using the web interface

DISPLAY RESULTS

And frames are displayed piece by piece
